Gefeliciteerd! De code staat op GitHub. Volg nu deze stappen:

### Stap 1: De Build Starten (Indien niet automatisch)
1. Ga naar je GitHub repository.
2. Klik op het tabblad **"Actions"**.
3. Selecteer **"APK-Forge-Build"** aan de linkerkant.
4. Als er geen actieve build is, klik op **"Run workflow"**.

### Stap 2: Download Link Genereren
Omdat GitHub artifacts standaard achter een login zitten, hebben we onderaan de app een **"Kopieer Download Link"** knop toegevoegd. 
Deze maakt gebruik van `nightly.link`, waardoor je de APK direct op je telefoon kunt downloaden zonder in te loggen op GitHub.

### Stap 3: Installeren
1. Open de gekopieerde link op je Android telefoon.
2. Download het .zip bestand en pak de .apk uit (of gebruik de directe .apk link indien beschikbaar).
3. Sta "Installeren van onbekende bronnen" toe in je Android instellingen.

**Let op:** Het duurt ongeveer 3 tot 5 minuten voordat de eerste build klaar is. De download link werkt pas nadat de eerste build succesvol is afgerond op GitHub!
